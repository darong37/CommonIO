# 変更仕様書：dp API 追加 と log STDERR 出力修正

日付：2026-04-19

## 変更スコープ

1. `_print_console_line` の廃止（バグ修正）
2. `dp` API の新規追加
3. `cpanfile` の新規作成（Data::Printer 依存追加）

---

## 1. `_print_console_line` の廃止

### 問題

`_print_console_line` は内部で `binmode STDERR, ':raw'` を実行していた。
これにより `BEGIN` ブロックが設定した STDERR のエンコード層が毎回 `log` 呼び出しのたびに破壊されていた。

### 修正内容（実施済み）

- `_print_console_line` サブルーチンを削除した。
- `log` 内の `_print_console_line($line)` 呼び出しを `print STDERR $line;` に置き換えた。
- エンコード変換の責務は `BEGIN` ブロックが設定した Perl のエンコード層に委ねる。

### 前提

- `use CommonIO` 時に `BEGIN` ブロックが1回だけ自動実行され、`I18N::Langinfo` の `langinfo(CODESET)` に従って `STDOUT` と `STDERR` のエンコーディングを設定する。利用者が明示的に呼ぶ必要はない。
- `log` はこの `BEGIN` ブロックが設定したエンコード層を前提として動作する。

## 2. `dp` API の追加

### インターフェース

```
dp(@args)
```

- 戻り値：なし
- 出力先：STDERR（`BEGIN` ブロックが設定したエンコード層経由）
- `@EXPORT_OK` に追加する

### 引数処理ルール

| 呼び出し | 内部処理 | 備考 |
|---|---|---|
| `dp()` | 何もしない | 引数なしは無視 |
| `dp($scalar)` | `np([@_])` | 非リファレンスは配列リファレンスにまとめる |
| `dp(\@array)` | `np(\@array)` | リファレンス1つはそのまま渡す |
| `dp(\%hash)` | `np(\%hash)` | リファレンス1つはそのまま渡す |
| `dp(@array)` | `np([@array])` | 複数引数は配列リファレンスにまとめる |
| `dp(%hash)` | `np([@_])` | 複数引数（偶数）も配列リファレンスにまとめる |

判定ロジック：
- 引数が0個 → 何もしない
- 引数が1個かつリファレンス → そのまま `np` に渡す
- それ以外（複数引数、または1個のスカラ） → `[@_]` で配列リファレンス化して渡す

### Unicode 表示設定

Data::Printer を `escape_chars => 'none'` で設定し、内部文字列をエスケープせず生の Unicode（漢字等）で表示する。

### 出力エンコーディング

`np()` の戻り値は Perl の内部文字列（utf8 フラグ ON）である。  
根拠：`perl -Mutf8 -MData::Printer -e 'my $x = "日本語"; my $s = np($x); print utf8::is_utf8($s) ? "yes" : "no"'` → `yes`、`length($s) == 5`、`length(encode("utf-8", $s)) == 11`。

`dp` は `np()` の戻り値を `print STDERR $out` でそのまま出力し、`BEGIN` ブロックが設定した STDERR のエンコード層に変換を委ねる（`log` と同じ方式）。raw ハンドルは使わない。

### 依存関係

- `use Data::Printer escape_chars => 'none'` を追加する。

## テスト方針

### `log` 回帰テスト

- 既存の `log` テストがそのまま通ることを確認する（新規テスト不要）。

### `dp` テスト

分岐ロジックの正しさを STDERR キャプチャで検証する。

| テストケース | 確認内容 |
|---|---|
| `dp()` | 例外が発生しない・何も出力しない |
| `dp({a => 1})` | リファレンス1個はそのまま渡る → 出力にハッシュ記法 `{` が含まれる |
| `dp('hello')` | 非リファレンスは `[@_]` に包まれる → 出力に配列記法 `[` が含まれる |
| `dp(1, 2, 3)` | 複数引数は `[@_]` に包まれる → 出力に配列記法 `[` が含まれる |
| `dp(['日本語'])` | 漢字を含む arrayref が例外なく出力される |
